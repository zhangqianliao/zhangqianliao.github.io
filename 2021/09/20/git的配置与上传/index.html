git的配置与上传

2021-09-20
配置git

一、下载并安装git（网速慢的同学请移步国内镜像）
二、配置git，使用两条命令
$ g --global user.name "输入GitHub用户名"
$ git config --global user.email "输入GitHub邮箱"
三、配置GitHub连接远程库
使用命令 $ ssh-keygen -t rea -C "输入GitHub邮箱"
四、登录GitHub使用生成的密钥进行创建库
使用git上传到GitHub

基本上传命令
|1|$ git status(查看状态)
|2|$ git add .(将新增的文件添加到代码仓库)
|3|$ git commit -m "提交说明"(提交到本地库)
|4|$ git push origin master(origin本地库master远程库push提交)
clear——张千燎(参考潘老师尚未配图)
