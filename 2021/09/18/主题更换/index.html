2021-09-18
hexo 更换主题步骤
在网上寻找自己喜欢的主题 例如hexo官网上的主题
1、将github上的主题资源下载到自己的项目中

例子 ：

$ git clone https://github.com/zhwangart/hexo-theme-ocean.git themes/ocean

    |1|git clone    拷贝git仓库到本地项目(简称下载)

    |2|https://github.com/zhwangart/hexo-theme-ocean.git   指的是 git仓库的地址

    |3|themes/ocean    主题名称

2、更改_config.yml中的主题    theme:主题hi名称    url:博客访问地址

3、使用hexo clean命令清除清除缓存文件 (db.json) 和已生成的静态文件 (public)。
在某些情况（尤其是更换主题后），如果发现您对站点的更改无论如何也不生效，您可能需要运行该命令

4、使用hexo generate命令生成静态页面

5、使用hexo deploy命令上传到github

        ——张千燎
