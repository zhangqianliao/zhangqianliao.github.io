git的配置与上传

2021-09-20
配置git

一、下载并安装git（网速慢的同学请移步国内镜像）
二、配置git，使用两条命令
$ g --global user.name "输入GitHub用户名"
$ git config --global user.email "输入GitHub邮箱"
三、配置GitHub连接远程库
使用命令 $ ssh-keygen -t rea -C "输入GitHub邮箱"
四、登录GitHub使用生成的密钥进行创建库
使用git上传到GitHub

基本上传命令
|1|$ git status(查看状态)
|2|$ git add .(将新增的文件添加到代码仓库)
|3|$ git commit -m "提交说明"(提交到本地库)
|4|$ git push origin master(origin本地库master远程库push提交)
clear——张千燎(参考潘老师尚未配图)

主题更换

2021-09-18
hexo 更换主题步骤
在网上寻找自己喜欢的主题 例如hexo官网上的主题
1、将github上的主题资源下载到自己的项目中

例子 ：

$ git clone https://github.com/zhwangart/hexo-theme-ocean.git themes/ocean

    |1|git clone    拷贝git仓库到本地项目(简称下载)

    |2|https://github.com/zhwangart/hexo-theme-ocean.git   指的是 git仓库的地址

    |3|themes/ocean    主题名称

2、更改_config.yml中的主题    theme:主题hi名称    url:博客访问地址

3、使用hexo clean命令清除清除缓存文件 (db.json) 和已生成的静态文件 (public)。
在某些情况（尤其是更换主题后），如果发现您对站点的更改无论如何也不生效，您可能需要运行该命令

4、使用hexo generate命令生成静态页面

5、使用hexo deploy命令上传到github

        ——张千燎
